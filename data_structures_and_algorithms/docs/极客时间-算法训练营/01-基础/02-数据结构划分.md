### 数据结构划分

1. 一维数据结构

   - 基础：

     - 数组 Array
     - 链表 Linked Array

   - 高级：
     - 栈 Stack
     - 队列 Queue
     - 双端队列 Deque
     - 集合 Set
     - 映射 Map
     - etc.

2. 二维数据结构

   - 基础：

     - 树 Tree
     - 图 Graph

   - 高级：
     - 二叉搜索树 Binary Search Tree
     - 平衡树 AVL
     - 红黑树 Red-Black Tree
     - 堆 Heap
     - 并查集 Disjoint Set
     - 字典树 Trie
     - etc.

3. 特殊数据结构

   - 位运算 Bitwise
   - 布隆过滤器 Bloom Filter
   - 缓存 LRU Cache
